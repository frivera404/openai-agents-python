Product: Accessible Interior Design — AI Prompts & Templates

Contents
- product_creator_output.pdf (generated by `assemble_pdf.py`)
- assemble_pdf.py — script to generate the PDF (uses ReportLab)
- requirements.txt — minimal requirements

How to build PDF locally
1. Create and activate a Python 3.10+ virtual environment.

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python assemble_pdf.py
```

Output: `product_creator_output.pdf` in the same folder.

Notes
- The script programmatically generates 1000+ prompts and draws simple vector illustrations and a mind map.
- If you want richer illustrations, replace the draw_illustration_page function or supply PNG/SVG assets and modify the script to embed them.
